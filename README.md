# Solutions to Rustlings Exercises

This repository contains my solutions to the exercises in the [Rustlings](https://rustlings.rust-lang.org/) course. The Rustlings course is a collection of small exercises that help you learn Rust by writing code. Each exercise focuses on a specific aspect of the Rust programming language.

I am trying to solve a few exercises each day, and I will be updating this repository with my solutions as I complete them.

Upon completion, I will hopefully find some starter project to complete or contribute to with my newfound knowledge of Rust.
